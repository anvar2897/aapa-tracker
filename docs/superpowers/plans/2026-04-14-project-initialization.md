# AAPA Tracker — Project Initialization Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Bootstrap Next.js 14 App Router project with Drizzle/SQLite schema (10 tables), shadcn/ui dark theme, and a realistic 7-product seed for the AAPA Uzum marketplace tracker.

**Architecture:** Bare Next.js project initialized in-place, Drizzle ORM with better-sqlite3 for a file-based database at /data/aapa-tracker.db, seed script runs at build time via `npx tsx`.

**Tech Stack:** Next.js 14, TypeScript strict, Tailwind CSS, shadcn/ui (Radix), better-sqlite3, Drizzle ORM + drizzle-kit, Recharts, tsx (for running TS scripts)
